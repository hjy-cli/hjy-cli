# hjy-cli(前端脚手架集合)
包含常用的项目框架

# 安装
```
npm install hjy-cli -g
```

# 使用
打开终端，输入“hjy”或“hjy -h”，您将看到下面的帮助信息:
```
  使用: hjy <命令>

  命令:

    add       添加一个新的脚手架模板
    delete    删除一个脚手架模板
    list      列出所有脚手架模板
    init      生成一个新项目

  选项:

    -h, --help     输出使用信息
    -V, --version  输出版本号
```

> 注意，如果您使用的是“MacOS”，那么在使用“add”和“delete”命令时，就需要使用“sudo”。
